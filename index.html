Welcome to my website! Below you will find my personal portfolio of all the projects I have worked on over the past few years. Each project features a brief description along with pictures and a link to the project.

AptiStudy

AptiStudy is a site that intends to augment and incentivize social opportunities through a more rewarding way with the application of study groups. Students can find various study groups, set up meetings for locations on campus, and complete a compatibility questionnaire to help students find study groups relating to their individual needs. Every web page was created in a PHP format to store user sessions and grab various dynamic information from our database using SQL queries.

Other contributors: Alex McGaugh, Josh Hatfield, and Jonathan Yen Tools/Languages: HTML, CSS, JavaScript, SQL, PHP

Sales Analysis Project

For this project I created a mock up business demand requesting sales, budget, and customer information from a companies database. The manager in this situation wanted specific information and a dynamic visualization with filtering capabilities. I created different SQL queries to grab the requested information and then imported the results into Power BI. From there I made dataset connections on the backend in Power BI. The dynamic dashboards I created are dynamic and display different information depending on the dates selected.

Tools/Languages: SSMS, SQL, Power BI

Johnson's Plant Site

Johnson's Plant Shop is a mobile-responsive website that showcases the captivating world of plants. It features a modern design and has a navigation bar that utilizes JavaScript. This site employs HTML semantics to ensure a structured and accessible user experience. You can find custom font choices and a carousel element featuring photos of my own plant collection.

Tools/Languages: HTML, CSS, JavaScript
